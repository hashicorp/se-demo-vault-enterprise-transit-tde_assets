Transparent Data Encryption (TDE)

This exercise explores the core benefits of using Vault Enterprise to secure data in Microsoft SQL Server using a defense-in-depth strategy to protect data at rest with transparent data encryption (TDE).

Vault Enterprise users can establish a key-management solution to manage the lifecyle of data encryption keys, which in turn protect data in SQL servers.

There are four micro-scenarios in this exercise. We aim to maintain a minimalist design principle so that each scenario can be flexible and adaptable with your own perspective.
